About

VLI@SUSS is Virtual Learning Infrastructure at Singapore University of Social Science. Our mission is to bring virtual computing & interactive digital experience to every classroom of SUSS and beyond.

Git & Version Control

Git is a free, open-source distributed version control system that tracks project and file changes with multiple contributors.

Linux

Linux is an open-source, Unix-like operating system for various devices developed by the community.

Docker

Docker is an open platform for developing, shipping, and running applications, empowering learners to run their cloud computing platform locally for optimal learning.

Cloud Computing & Infrastructure

Cloud computing delivers computing services over the Internet, providing faster innovation, flexible resources, and economies of scale.

Interactive Lab Guide

We believe an interactive lab guide is the best tool for engaging learners in a smart way.

Autograding, Autotesting & Automation

We believe that machine-gradable exercises/questions gives learners continuous feedback on their coding assignments.
